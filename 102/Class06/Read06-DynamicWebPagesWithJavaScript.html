Dynamic web pages with JavaScript

JavaScript

JavaScript (JS) is a lightweight, interpreted, or just-in-time compiled programming language with 1st class functions mostly known as a scripting language for Web pages, but Node.js, Apache, CouchDB, and Adobe Acrobat all use it too
traditionally JavaScript was put inside web browsers but now people put it on the server 2, it would just b included in the HTML page the user received when they visit a web site & run in the browser from the client side, as opposed to running on the web server side like they do nowadys

there r 3 major parts of JavaScript: The language itself. This is fairly standard among the various environments, both in the various browsers and in the various server-side environments.
The DOM API - how the language can interact with the various parts of a web page while in the browser. While in this respect the various browsers are getting closer to each other they still differ. Several libraries, most prominently JQuery, is trying to provide a unified API.
The server API provided by Node.js or one of the other server-side systems.

Embed or Include

u can either embed JavaScript directly inside the HTML file, or u can put a line in the HTML file that includes the external JavaScript file the latter is recommended 2 embed it directly in your html add the <script>opening/closing tags</script> & write ur JavaScript between them.

Input Output

There r lots of ways JS can display text for the user (output) 1 way is with the alert function

alert

Alert triggers a pop-up in browser with text, while rarely used it is an easy way to show the use of JavaScript <script language="javascript"> alert("Hello World"); </script>

Prompt

prompt shows a pop-up window with the text provided as the 1st parameter + a textbox u can fill in, when the user presses OK the value in the text box is returned by the prompt function. <script> var name = prompt("Your name:", ""); document.write("Hello ", name); </script>

Confirm

The other pop-up allows u to ask Yes/No questions, Calling the confirm function will show a pop-up window with the provided texts and 2 buttons. If the user presses OK the confirm function will return true, if the user presses cancel or hits the ESC key the function will return false. <script> if (confirm("Shall I print Hello World?")) { document.write("Hello World"); } else { document.write("OK, I won't print it."); } </script>
