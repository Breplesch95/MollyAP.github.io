HTML: HyperText Markup Language

HTML is the most basic building block of the internet, it defines the meaning and structure of the content on it, there are other technologies too, like css which describes a web page's appearance/presentation and JavaScript which describes it's functionality/behavior

The "Hypertext" in HTML refers to links that connect web pages, either within a single website or between websites, links are a fundamental aspect of the Webwhen you upload content to the Internet and link it to pages created by other people, you become an active participant in the World Wide Web The "markup" in HTML refers to it annotating text, images, and other content for display in a Web browser. HTML markup includes special "elements" such as:

< head > - container for metadata (data about the HTML document), and thus is not displayed, typically defines the documents title, character set, styles, scripts, and other meta information

< title > - text-only element that defines the title of the document, and is shown in the browser's title bar or in the page's tab (very important for search engine optimization aka SEO)

< body > - defines the document's body, consequently containing all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

< header > - container for introductory content or a set of navigational links typically containing one or more heading elements (< h1 > - < h6 >), logo/icon, or authorship information

< footer > - defines the bottom of a web page/section, usualy containing copyright info, navigation links, contact details, etc, but in the case of the last one, put it in the "< address >" tag

< article > - independent, self-contained content that should make sense on its own and be possible to distribute independently from the rest of the site.

< section > - standalone sections on a page with logically connected content (news block, contact information), often used when creating a landing page to divide it into separate logical blocks.

< p > - defines a paragraph, always starts on a new line and browsers automatically add a margin before and after each one, but if you just need to move text to a new line use the

For example, the tag can be written as <Title>, <TITLE>, or in any other way.
